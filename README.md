## 1. 기획 목적 🔥
- 단순 판매 커뮤니티가 아닌 공예품에 관심있는 모든 사람들의 커뮤니티로서 회원간 소통과 클래스 예약을 동시에 할 수 있는 멀티 플랫폼

- 회원 각각의 소셜 커뮤니티를 생성하여 회원들 간의 소통 기능 제공
